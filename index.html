Hi, I'm James Thompson

I'm a Front End Web Developer based in London.

I specialise in HTML, CSS, JavaScript and React.

About Me

I originally have a background in both finance and law, but in early 2022 I decided that I wanted a career change. Consequently, I started learning to code in February. Initially, I began by learning the basics of Python, since I'd read online that Python had the easiest syntax for beginners to understand and that there was a lot of free information to help me if I got stuck on a problem.

This experience in Python led to me being accepted into a front end web development bootcamp run by Bath Spa University. This bootcamp ran from early March to late May 2022 and taught me the fundamentals of HTML, CSS JavaScript and React.

Using the skills learned during my time at Bath Spa University, I then managed to secure a place at Futureproof - a technology training company which runs its own bootcamp for successful applicants and then secures a two-year commercial placement for people who successfully graduate.

Overall, I would say I am learning a lot and improving rapidly. If my profile interests you, I would recommend that you look at my work-related social media (LinkedIn and GitHub) or download my CV.

Projects

Form Validation Project

This was a great project for developing and applying my knowledge of regular expressions to validate form entries. This project also boosted my knowledge of writing complicated, lengthy functions and provided me with the opportunity to use multiple for each loops.

Starbucks Landing Page

This was a great project for trying to create a landing page that a big company would potentially use. This landing page is responsive for both mobile and desktop devices, although it still needs work for users on tablets.

Bio Page

This was a great project for learning about how to apply a dark mode to the elements in the webpage. Further, it is fully responsive regardless of the screen size that the end user is using. Finally, it has a little true or false quiz which was useful for learning about forms.

Accordion Project

This was a great project for learning about the JavaScript that is necessary to build an accordion. Rather than using CSS, I used JavaScript to apply a transform property to the carat that rotates it 180 degrees and reveals the hidden text. If clicked again, it reverts back to hidden.

Counter Project

This was a great project for learning about event listeners. Not only are there click event listeners for the buttons, there are also event listeners added to the window which allow the user to change the number by pressing either the up or down arrow key.

Slideshow Project

This was a great project for learning how to create an interactive carousel where only one child element within the carousel is visible at any given time. It also strengthened my knowledge of how to use event listeners (including on the window).

Random Colour Generator

This was a great project for strengthened my knowledge of: for loops, while loops, if else statements and functions. Also, it taught me to use the clipboard API to copy the inner text of an HTML element.

Car Card Project

Another Frontend Mentor Newbie Challenge which involves HTML and CSS. This challenge provided an interesting opportunity to apply media queries which alter the layout depending on the screen size.

Stats Preview Card

Another Frontend Mentor Newbie Challenge which involves HTML and CSS. This challenge provided an interesting opportunity to apply media queries which alter the layout depending on the screen size. This also taught me about the CSS property mix-blend-mode.

Interactive Rating Component

Another challenge which was a nice way to introduce manipulating the DOM. I wrote the javascript in such a way that irrespective of how many times the end user changes their mind the most recent click is their rating.

Five Star Rating System

This project makes good use of for loops to highlight all previous stars up to star that user clicks. Also makes good use of mouseenter and mouseleave event listeners to ensure hover classes remain even after user has clicked stars.

Weather API Project

This was my first major project working with an API. This project introduced me to the fetch method as well as the then method. Unfortunately, I have no live website for this project since I have not learned how to hide my API token from the user (yet).
